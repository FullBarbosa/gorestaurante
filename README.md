<div id="dependencias">
<h3>Baixar Projeto.</h3>  

```bash
# Clone this repository
$ git clone https://github.com/FullBarbosa/gorestaurante.git

# Install dependencies Yarn
$ yarn install

# Install dependencies NPM
$ npm install 
```
</div>

<div>

<h3>Como utilizar.</h3>  

```bash
# Start do servidor

$ yarn server

# Iniciar aplicação

$ yarn start

```
